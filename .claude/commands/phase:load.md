# Load Phase

Use this phase to understand previous work before starting a new cycle.

## Grounding

- Read `AGENTS.md` and `docs/README.md`.
- Read `.agent/phase/summary.md` if it exists.
- Do not implement code in this phase.

## Output

Summarize:
- What was already built.
- Important architecture decisions.
- Validation results and known limitations.
- Likely integration points for the next task.

Then guide the user to `/phase1:brainstorm`, `/phase2:plan`, or `/phase3:work`.
